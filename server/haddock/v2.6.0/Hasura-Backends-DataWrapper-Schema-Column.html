data Column Source #

A schematic representation which captures common attributes associated with a piece of data that is stored in a given backend.

These attributes ascribe meaningful semantics to the data that they are associated with.

cf. https://en.wikipedia.org/wiki/Column_(database) https://www.postgresql.org/docs/13/ddl-basics.html
