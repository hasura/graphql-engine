# EE: Introduction <HeadingIcon icon={`ee`} size={36} />

The following documentation pages describe the installation and setup for Hasura EE - Hasura Enterprise Edition.

To be able to install Hasura EE you will need a **service account token**.

Please contact our [sales team](mailto:sales@hasura.io) if you need help.

:::info Note

If you'd like to try Hasura without having to install Hasura EE, we recommend starting with the Hasura Community Edition
using Docker or on Hasura Cloud. Visit the [getting started guides](/getting-started/index.mdx).

:::
