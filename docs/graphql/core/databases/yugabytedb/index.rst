Introduction
------------

`YugabyteDB <https://www.yugabyte.com/>`__ is the resilient, scalable, high 
performance distributed SQL database. It is 100% open source, PostgreSQL-compatible, 
and designed to run on any cloud. As it is feature compatible with PostgreSQL, all 
of PostgreSQL's features, tooling, and ecosystem can be leveraged.
