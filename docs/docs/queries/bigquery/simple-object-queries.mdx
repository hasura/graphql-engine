# BigQuery: Simple Object Queries

## Introduction

You can fetch a single node or multiple nodes of the same type using a simple object query.

## Fetch list of objects

**Example:** Fetch a list of authors:

<GraphiQLIDE
  query={`query {
  bigquery_authors {
    id
    name
  }
}`}
  response={`{
  "data": {
    "bigquery_authors": [
      {
        "id": "1",
        "name": "Justin"
      },
      {
        "id": "2",
        "name": "Beltran"
      },
      {
        "id": "3",
        "name": "Sidney"
      },
      {
        "id": "4",
        "name": "Anjela"
      }
    ]
  }
}`}
/>

:::info Note

The query field will be of the format `<dataset_name>_<table_name>`.

:::

See the [API reference](/api-reference/graphql-api/query.mdx#simpleobject) for more details.
