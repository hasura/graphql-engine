# Marketplaces

<div className="badge badge--primary heading-badge">Available on: Cloud Enterprise</div>

## Overview

Hasura allows you to pay for Hasura Cloud Enterprise via private offers in the
[AWS Marketplace](https://aws.amazon.com/marketplace/pp/prodview-h7qmhw3draxh4?sr=0-1&ref_=beagle&applicationId=AWSMPContessa)
as well as the
[GCP Marketplace](https://console.cloud.google.com/marketplace/product/hasura-public/hasura-enterprise?project=hasura-public).
To sign-up or for more details, please contact your sales representative or [sales](mailto:sales@hasura.io).
