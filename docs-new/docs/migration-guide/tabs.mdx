# Tabs
Should be a straight forward syntax replacement

<Tabs>
  <TabItem value="sphinx" label="Sphinx - RST" default>

```rst
  .. rst-class:: api_tabs
  // highlight-next-line
  .. tabs::

    // highlight-next-line
    .. tab:: Console

      Click the ``Reload`` button in the ``Remote schema`` section on the Hasura console.

    // highlight-next-line
    .. tab:: API

      Make a request to the :ref:`reload_remote_schema<reload_remote_schema>` API.
```

  </TabItem>
  <TabItem value="docusaurus" label="Docusaurus - MDX">

```jsx
  import Tabs from '@theme/Tabs';
  import TabItem from '@theme/TabItem';

  <Tabs className="api-tabs">
    <TabItem value="console" label="Console">
      Click the <code>Reload</code> button in the <code>Remote schema</code> section on the Hasura console.
    </TabItem>
    <TabItem value="api" label="API">
      Make a request to the <a href="#">reload_remote_schema</a> API.
    </TabItem>
  </Tabs>
```
  </TabItem>
</Tabs>
