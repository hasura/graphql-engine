### `2.0.9`

- We now support exporting Hasura metrics in Prometheus format via a specialized endpoint. This allows you to use a
  Prometheus agent to scrape and load your metrics into a Prometheus compatible for further analysis. Config
  [here](https://docs.ee.hasura.io/lux-installation/helm-configs/#configure-prometheus-exporter)

- The Hasura control plane service has been upgraded to 2.x.x and you can now disable introspection queries
- The control plane dashboard now supports the ability to list projects across multiple pages
- Compatible with graphql-engine-pro versions 1.3.x and 2.0.x
