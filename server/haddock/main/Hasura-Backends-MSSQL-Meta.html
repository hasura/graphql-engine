Description

Metadata related types, functions and helpers.

Provides a single function which loads the MSSQL database metadata. See the file at src-rsr/mssql_table_metadata.sql for the SQL we use to build this metadata. See DBTableMetadata for the Haskell type we use forall storing this metadata.
