This tutorial is for android developers to learn what GraphQL is and how to use GraphQL in android applications.
It is ideal if you follow along with code and should not take you more than a few hours to finish end to end.

## What do I need to take this tutorial?

Your dev environment set up with Android Studio to build android apps. If you can create an android app from Android Studio and run it on emulator, you're good to go!

## How long will this tutorial take?

Less than 2 hours

## What you'll be able to do after this tutorial

Use (realtime) GraphQL in your android app.

## Who is this tutorial for

You should know the basics of how to build a android app.

## Topics covered in this tutorial

- Introduction to GraphQL
- Using Apollo client
- GraphQL queries
- GraphQL mutations & variables
- GraphQL subscriptions
- Making direct GraphQL API calls without Apollo's higher-order-components
