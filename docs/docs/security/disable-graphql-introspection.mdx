# Cloud & EE: Disable GraphQL introspection <HeadingIcon icon={`cloud-ee`} size={36} />

## Introduction

The GraphQL engine by default enables
[GraphQL Schema Introspection](https://spec.graphql.org/June2018/#sec-Schema-Introspection) which gives full information
about the schema and may not be desired in production.

To avoid this, you can disable GraphQL introspection on a per-role basis.
