Schema Metadata API Reference
==================

The Schema Metadata API provides the following features:

1. Execute SQL on the underlying Postgres database, supports schema modifying actions. (``admin`` only)
2. Modify Hasura metadata (permissions rules and relationships) (``admin`` only)
