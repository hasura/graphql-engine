# MS SQL Server: Simple object queries

## Introduction

You can fetch a single node or multiple nodes of the same type using a
simple object query.

## Fetch list of objects

**Example:** Fetch a list of authors:

<GraphiQLIDE
  query={`query {
  authors {
    id
    name
  }
}`}
  response={`{
  "data": {
    "authors": [
      {
        "id": 1,
        "name": "Justin"
      },
      {
        "id": 2,
        "name": "Beltran"
      },
      {
        "id": 3,
        "name": "Sidney"
      },
      {
        "id": 4,
        "name": "Anjela"
      }
    ]
  }
}`}
/>
