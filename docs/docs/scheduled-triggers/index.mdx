# <ScheduledTriggers /> Scheduled Triggers

## Introduction

Scheduled Triggers are used to execute custom business logic at specific points in time.

<Thumbnail
  src="/img/scheduled-triggers/scheduled-trigger-arch.png"
  alt="Hasura Scheduled Trigger architecture"
  width="700px"
  className="no-shadow"
/>

:::tip Supported from

Scheduled Triggers are supported from versions `v1.3.0` and above.

:::

## Types

There are two types of timed events:

- [Cron triggers](/scheduled-triggers/create-cron-trigger.mdx): based on a cron schedule.
- [One-off scheduled events](/scheduled-triggers/create-one-off-scheduled-event.mdx): based on a fixed timestamp.
