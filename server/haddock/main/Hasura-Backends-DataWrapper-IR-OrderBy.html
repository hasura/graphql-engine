data OrderBy Source #

Indicates a particular sort order that should be applied based on some Name returned within a data source query.

TODO: We should use a sum type like Query.Field here so that we can handle order by constraints on object/array relationships as well.

cf. https://www.postgresql.org/docs/13/queries-order.html
