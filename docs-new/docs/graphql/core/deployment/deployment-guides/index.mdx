# Deployment guides

## Introduction

The Hasura GraphQL engine is a binary that is shipped as a Docker
container.

Choose from the below guides to deploy the Hasura GraphQL engine and
connect it to a Postgres database.

## One-click deployment options {#one-click-deployment-guides}

If you want to take Hasura for a spin and check out the features, the
following one-click deployments will be useful for you:

- [Deploy using Hasura Cloud](/graphql/cloud/getting-started/index.mdx) (**recommended**)
- [Deploy using Heroku](/graphql/core/deployment/deployment-guides/heroku.mdx#heroku-one-click)
- [Deploy using Render One-click Deploy with Managed PostgreSQL](/graphql/core/deployment/deployment-guides/render-one-click.mdx)
- [Deploy using Nhost One-click Deploy with Managed PostgreSQL, Storage, and Auth](/graphql/core/deployment/deployment-guides/nhost-one-click.mdx)
