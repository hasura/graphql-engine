Getting started with YugabyteDB
=======================================

To try Hasura with YugabyteDB, you'll need your own new or existing YugabyteDB cluster. To create 
a YugabyteDB cluster:

* You can `sign up for Yugabyte Cloud <https://cloud.yugabyte.com/register/>`__ and get a free YugabyteDB cluster.
* You can `install YugabyteDB <https://docs.yugabyte.com/latest/quick-start/install/>`__ on MacOS, Linux, Docker, or Kubernetes.

Here are 2 ways you can get started with Hasura:

1. :ref:`Hasura Cloud <database_yugabytedb_cloud>`: You'll need to be able to access your YugabyteDB cluster from Hasura Cloud.
2. :ref:`Docker <database_yugabytedb_docker>`: Run Hasura with Docker and then connect your YugabyteDB cluster to Hasura.
