# Hasura CLI

## Introduction

The `Hasura CLI` is a command line tool which is the primary mode of
managing Hasura projects and migrations.

## Installation

Refer to [Installing the Hasura CLI](/graphql/core/hasura-cli/install-hasura-cli).
